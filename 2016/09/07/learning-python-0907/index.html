【9.7】Python学习笔记-python基础

python基础

数据类型与变量

转义字符：\
默认不转义：r'\n\\'
字符串内部多行：

‘’’line1
line2
line3’’’

空值None
地板除，用//表示

>>>10//3

字符串和编码

ACSII，1个字节
Unicode，通常2个字节，存在服务器，内存中
UTF-8，英文1个字节，汉字通常3字节
获取字符串整数 

>>>ord('A')
65

编码转换成对应字符 

>>>chr(25991)
'文'

要在网络上传输，或者保存到磁盘上，就需要把str变为以字节为单位的bytes。 x = b'ABC' bytes的每个字符只占用一个字节。
以Unicode表示的str通过encode()方法可以编码为指定的bytes。

>>>'ABC'.encode('ascii')
b'ABC'

反过来，如果我们从网络或磁盘上读取了字节流，那么读到的数据就是bytes。要把bytes变为str，就需要用decode()方法：

>>>b'\xe4\xb8\xad\xe6\x96\x87'.decode('utf-8')
'中文'

计算字符长度：len('abc')
编码注释：#-*- coding:utf-8 -*-
格式化字符串：%d-整数，%f-浮点数， %s-字符串，%x-十六进制整数

>>>'%.2f' % 3.1415926
'3.14'

使用list和tuple

list

>>>list = ['0','1','2','3']
>>>list[0]
'0'
>>>list[-1]
'3'

添加元素：

>>>list.append('4')
>>>list
['0','1','2','3','4']

指定位置插入元素：

>>>list.insert(1,'4')
>>>list
['0','4','1','2','3']

删除末尾元素：

>>>list.pop()
>>>list
['0','1','2']

删除指定位置元素：

>>>list.pop(1)
>>>list
['0','2','3']

替换元素

>>>list[1] = '4' #赋值
>>>list
['0','4','2','3']

list里的数据类型可以不同如list = ['0',123,True]
一个list也可以作为一个元素s = ['C','C++',['Java','Python']],相当于

>>>p = ['Java','Python']
>>>s = ['C','C++',p]

若list中一个元素都没有，list = []，长度为0，len(list) = 0
tuple

tuple = ('0','1','2')

一旦初始化后不能修改，没有替换，删除等用法，获取元素方法相同如tuple[0] = '0'，因其不可更改所以代码更安全，尽量使用tuple
tuple的陷阱
定义空的tuple，写成

>>>t = ()
>>>t
()

定义只包含一个元素的tuple

>>>t = (1)
>>>t
1

这个是错误的，因为()既可以表示tuple，也可以表示数学意义上的小括号，python规定，按小括号计算，所以表示只包含一个元素的tuple需要加一个逗号

>>>t = (1,)
>>>t
(1,)

可变的tuple

>>>t = ('a','b',['A','B'])
>>>t[2][0] = 'X'
>>>t[2][2] = 'Y'
>>>t
('a','b',['X','Y'])

改变的只是tuple中的list，并不是tuple的元素（指向不变）
创建一个不变的tuple，就要保证tuple每个元素不可变
整数，浮点数，字符串都是不可变对象

list和tuple是python内置的有序集合，list可变，tuple不可变，分情况使用。

条件判断

if <条件判断1>:
    <执行语句1>
elif <条件判断2>:

for…in循环

>>>list(range(5))

while <条件判断>:
    循环体

使用dict和set

dict（字典，dictionary）
使用key-value存储，具有极快查找速度

>>>d = {'Tom':95,'Sam':75,'John':85}
>>>d['Tom']
95

将数据放入dict中，除了初始化外，也可以通过key放入

>>>d['Sam'] = 99
>>>d['Sam']
99

一个key只能对应一个value，多次对一个key放入value，会被覆盖
key不存在时，dict会报错，避免key不存在的检测方法：

1.通过in来判断key

>>>'John' in d
True

2.通过dict提供的get方法，不存在返回None，也可指定value

>>>d.get['Tom']
>>>d.get['Tom',90]
90

删除key，用pop(key)的方法

>>>d.pop('Tom')
95
>>>d
{'Sam':75,'John':85}

dict内部存放顺序和key放入顺序无关
key必须是不可变对象，list可变所以不可以作为key
set
set与dict类似，也是一组key的集合，但不存储value，且key不重复,重复元素被过滤

>>>s = set([1,2,3])
>>>s
{1,2,3}

通过add(key)添加元素
通过remove(key)删除元素
set可以看成数学意义上的无序且无重复元素的集合，因此两个set可以做数学意义上的交集和并集。
set也同dict一样，同样不可以放入可变对象，tuple虽然是不可变对象，但其元素可能包含可变对象list
不可变对象（str操作）

>>>a = 'abc'
>>>b = a.replace('a','A')
>>>b
'Abc'
>>>a
'abc'


对不可变对象来说，调用对象自身的任意方法，不会改变该对象自身的内容，这些方法只会创建新的对象并返回。
